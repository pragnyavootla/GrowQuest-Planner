# GrowQuest Planner

## Design Overview

GrowQuest Planner is a nature-inspired task management application designed to make productivity feel calm, rewarding, and visually engaging. Instead of using a traditional checklist approach, the planner represents progress through the growth of a plant, turning completed tasks into a visual journey from a seed to a blooming flower.

## HTML Structure

The application is built using semantic HTML and organized into clear sections:

* A header displaying the project title.
* A daily wellness reminder section.
* A plant growth area that visually reflects task completion progress.
* A progress bar showing overall completion percentage.
* Input fields for task creation and due dates.
* A dynamic task list where tasks are displayed and managed.

This structure keeps the interface intuitive while separating different functionalities into dedicated sections.

## Styling and Visual Design

The interface uses the **Comfortaa** font to create a soft, friendly, and approachable appearance. Its rounded letterforms complement the planner's relaxing and growth-focused theme.

The colour palette is inspired by nature and plant life:

* Soft greens represent growth and productivity.
* Light mint and pastel backgrounds create a peaceful atmosphere.
* Darker green accents highlight important interactive elements.
* A dedicated dark mode uses deep navy and forest-inspired tones to maintain readability while preserving the overall aesthetic.

Rounded corners, subtle shadows, and gentle gradients were used throughout the design to create a modern and welcoming user experience.

## Plant Growth Theme

A central feature of the planner is the plant growth system.

As tasks are completed, the plant progresses through multiple growth stages:

* 🌱 Seed
* 🌿 Sprout
* 🪴 Growing Plant
* 🌳 Mature Tree
* 🌲 Almost There
* 🌸 Blooming Flower

This visual feedback provides motivation by transforming productivity into a rewarding growth journey.

## Task and Due Date Management

Users can add tasks along with optional due dates. Each task is displayed with its associated deadline, making it easier to organize upcoming work.

The application automatically checks all tasks against the current date. If a task's due date has passed, it is marked as overdue through visual styling changes. This helps users quickly identify unfinished tasks that require attention.

## Progress Tracking

The progress system calculates the percentage of completed tasks relative to the total number of tasks.

As users check off completed tasks:

* The progress bar updates smoothly.
* The completion percentage is displayed in real time.
* The plant growth stage changes accordingly.

This creates a direct connection between task completion and visual progress, encouraging users to stay consistent and complete their goals.

## JavaScript Functionality

JavaScript powers the interactive features of the planner, including:

* Adding new tasks dynamically.
* Creating and managing due dates.
* Marking tasks as completed.
* Removing tasks from the list.
* Detecting overdue tasks automatically.
* Updating the progress bar and percentage.
* Managing plant growth stages.
* Switching between light mode and dark mode.

The application updates instantly without requiring page reloads, creating a smooth and responsive user experience.

## Conclusion

GrowQuest Planner combines productivity tools with a calming plant-growth concept. Through nature-inspired visuals, real-time progress tracking, due date monitoring, and interactive task management, the application transforms everyday planning into a more engaging and rewarding experience.
